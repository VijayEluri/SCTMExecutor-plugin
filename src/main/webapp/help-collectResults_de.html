Diese Option ermöglicht es das Sammeln der Testergebnisse zu aktivieren/deaktivieren.
Notiz:Werden keine Testergebnisse eingesammelt, wird das folgende Build Verfahren (Build Trigger) unverzüglich angestoßen. Um Testergebnisse in Jenkins zu sehen muss eine Post-Build-Aktion "Collect Silk Central Test Results" konfiguriert werden .
